#unit-tracker

A small map application which tracks the paths of various units.

###Dependencies:
1. install [http://nodejs.org/](nodejs)
2. install [http://mimosa.io/](mimosajs)

   ```
    $ npm install -g mimosa
   ```
3. install all other dependencies:

   ```
   $ npm install
   ```

###Building:
```
$ mimosa build -o
```

###Run Revelopment:
```
$ mimosa watch --server
```
